Alteração status recurso

Nessa tela são realizadas as alterações de status do recurso.

Para acessar essa tela pode utilizar as opções:

  • Através do menu Produção -> Apontamentos -> Alteração status recurso;

  • Através do Terminal de Apontamentos;

Para alterar o status de um recurso, siga as instruções etapas:

  • Especifique o recurso que deseja modificar;

  • Especifique o novo status que deseja atribuir ao recurso;

  • Pressione o botão “Adicionar” para aplicar a alteração.

É importante ressaltar que apenas os recursos que não estão sendo utilizados em produção serão exibidos para seleção. Isso garante que você não faça alterações em recursos em uso, evitando possíveis interrupções indesejadas.

No entanto, se o usuário tentar alterar o status sem especificar o recurso, será exibida a seguinte mensagem:

BR One :: Informe um valor para o campo “Recurso”

Serão exibidos apenas os status que pertençam a alguma destas categorias:

-Fora de turno

-Parada não planejada

-Parada planejada I

-Parada planejada II

Caso o usuário tente alterar o status sem especificar o status, será exibida a seguinte mensagem:

BR One :: Informe um valor para o campo “Status”
