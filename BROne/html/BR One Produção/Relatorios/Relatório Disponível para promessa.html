Relatório Disponível para promessa

Este relatório realiza a compilação das informações das quantidades de estoque disponível para “Pedido”, “Confirmado” e “Disponível”, ele pode ser acessado pela tela de “Cadastro de item” basta clicar com o botão direito e na opção de “Quantidade disponível para vendas”.

Ao clicar como botão direito na tela de “Cadastro de item” e depois na opção “Quantidade disponível para vendas”, o relatório será carregado:

O relatório “Disponível para promessa” realiza os cálculos para as quantidades existentes de para três campos em especifico:

Pedido

Informa as quantidades dos documentos que garantem a disponibilidade do item no estoque, exemplo, Ordem de Produção.

Confirmado

Informa as quantidades dos documentos que confirmam o consumo do item, exemplo, Pedido de Venda.

Disponível

Realiza o cálculo das quantidades para o item:

  • Quando o cálculo é referente a coluna “Pedido”, é realizado uma soma.

  • Quando o cálculo é referente a coluna “Confirmado” é realizado uma subtração.
