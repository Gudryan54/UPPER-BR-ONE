Versão 4.11.330
~~~~~~~~~~~~~~~

.. important::

   **BR One API - Funcionalidades no Módulo de Produção e Qualidade**

   
   `Link do manual e guia completo de configuração <https://brone.help.uppertools.com.br/BROne/html/BR%20One%20API.html>`_

**Apontamento de Produção**

- **Busca de apontamentos de produção**

  Permitir buscar todos os apontamentos existentes na base.

- **Busca somente ordens de produção que podem receber apontamentos**

  Permite filtrar as ordens de produção que podem receber apontamentos.

- **Busca de operações que possam receber apontamentos**

  Filtra operações específicas para receber apontamento de horas dos recursos.

- **Busca de apontamentos conforme parâmetros especificados**

  Realiza busca conforme parâmetros especificados, garantindo maior precisão.

- **Adição de apontamento de produção**

  Possibilita adicionar novos apontamentos de produção com recursos e horas associadas.

- **Atualização de apontamento de produção**

  Permite atualizar registros específicos de apontamentos.

- **Exclusão de linhas em apontamentos de produção**

  Remove linhas específicas dentro de apontamentos existentes.

- **Exclusão de apontamento de produção**

  Elimina apontamentos de produção completos da base de dados.

- **Conclusão de apontamentos de produção**

  Finaliza um apontamento de produção, marcando-o como completo.

**Apontamento de Refugo**

- **Adição de apontamento de refugo**

  Possibilita a adição de registros de refugo em ordens de produção.

**Colaboradores**

- **Busca de colaboradores**

  Realiza uma busca na base de colaboradores para associá-los a diferentes processos.

**Entrada de Produto Acabado**

- **Adição de entrada de produto acabado**

  Permite registrar a entrada de produtos acabados na base.

- **Entrada de PA com desvio na OP de retrabalho**

  Registra entradas de produtos acabados que contêm desvios específicos em ordens de retrabalho.

**Ordem de Produção**

- **Busca de ordens de produção**

  Permite localizar ordens de produção ativas ou específicas.

- **Adição de uma ordem de produção**

  Insere uma nova ordem de produção com dados completos.

- **Adição de linha no GRID da ordem de produção**

  Adiciona uma nova linha de informações ao GRID de uma ordem.

- **Exclusão de linha na ordem de produção**

  Remove linhas específicas de uma ordem de produção.

- **Liberação de ordens de produção**

  Libera ordens para o próximo estágio de processamento.

- **Atualização de ordens de produção**

  Realiza atualizações em ordens existentes.

- **Fechamento de ordens de produção**

  Encerra ordens de produção informadas; atualmente, os tipos suportados incluem:
  
  - OP Padrão
  - OP de Retrabalho
  - OP de Beneficiamento

**Status dos Recursos**

- **Busca status dos recursos**

  Exibe o status atual dos recursos para acompanhamento e controle.

**Classificações de Refugo**

- **Busca classificações de refugo**

  Permite consultar as categorias de refugo disponíveis para organização dos registros.

**Saída de Insumos**

- **Adição de saída de insumos**

  Registra a saída de insumos no sistema.

- **Adição de saída de insumos com itens alternativos**

  Permite saída de insumos utilizando itens alternativos conforme necessário.

**Ficha de Análise**

- **Busca de fichas de análise por Nº de análise**

  Localiza fichas específicas pelo número da análise.

- **Busca de fichas de análise por DocNum**

  Permite a busca de fichas usando o DocNum.

- **Atualização da ficha de análise**

  Permite atualizar informações na ficha de análise.

- **Atualização do status da ficha de análise**

  Modifica o status atual da ficha de análise.

- **Atualização da quantidade do palete na ficha de análise**

  Ajusta a quantidade do palete especificada em uma ficha de análise.
