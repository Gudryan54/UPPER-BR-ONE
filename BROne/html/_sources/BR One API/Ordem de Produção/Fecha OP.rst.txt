Fechar Ordens de Produção
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este endpoint realiza o fechamento das ordens de produção informadas no campo '**docNumOrdemProducao**', atualmente é possível fechar os seguintes tipos de OPs:

- OP Padrão
- OP de Retrabalho
- OP de Beneficiamento
