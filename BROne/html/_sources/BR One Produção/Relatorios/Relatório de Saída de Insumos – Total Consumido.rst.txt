Relatório de Saída de Insumos – Total Consumido
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório retorna uma lista de todas as saídas de insumos realizadas, o retorno é realizado de acordo com os filtros selecionados.

.. image:: /_static/BR\ One\ Produção/Relatorios/Saída\ de\ Insumos\ total\ consumido/29.png

- **Data Saída Inicial:** Define a partir da data para buscar e realizar as analises para o Relatório  
- **Data Saída Final:** Define até a da data para buscar e realizar as analises para o Relatório  
- **Ordem Inicial:** Define a partir de qual OP será considerada para analise para ser levado ao relatório 
- **Ordem Final:** Define até a OP que será considerada para analise para ser levado ao relatório 
- **Item PA:** Define o Item que será analisado.
