Relatório Entrada de Produto Acabado
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório retorna uma lista de todas as entradas de PA realizadas nas OPs, o retorno é realizado de acordo com os filtros selecionados.

.. image:: /_static/BR\ One\ Produção/Relatorios/Entrada\ de\ Produto\ Acabado/31.png

- **Inserir Item PA:** Define seleção de PAs que serão retornados.
- **Inserir depósito:** Define depósito que será retornado.
- **E/S:** Define se retorna as Entradas ou os Estornos.
