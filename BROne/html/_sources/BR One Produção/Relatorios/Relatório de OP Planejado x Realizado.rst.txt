Relatório de OP Planejado x Realizado
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório retorna uma lista comparando o apontamento Planejado x Realizado para a OP, o retorno é realizado de acordo com os filtros selecionados.

.. image:: /_static/BR\ One\ Produção/Relatorios/OP\ Planejado\ x\ Realizado/17.png

- **Número da OP:** Define a OP a ser analisada.
- **Item Produzido:** Define o PA, de acordo com a OP escolhida
- **Data Liberação De:** Define a data de liberação inicial.
- **Data de Liberação Até:** Define a data de liberação final
