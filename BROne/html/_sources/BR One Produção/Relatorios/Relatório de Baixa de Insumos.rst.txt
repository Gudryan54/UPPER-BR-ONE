Relatório de Baixa de Insumos
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório retorna uma lista das saídas de insumos realizadas para os componentes das OPs, de acordo com os filtros selecionados.

.. image:: /_static/BR\ One\ Produção/Relatorios/Baixa\ de\ Insumos/15.png

- **Ordem de Produção De:** Define a partir da Ordem de Produção (OP) selecionada para a análise.
- **Ordem de Produção Até:** Define até a Ordem de Produção (OP) selecionada para a análise.
- **Item Pai De:** Define o início do intervalo para a análise do item selecionado.
- **Item Pai Até:** Define o término do intervalo para a análise do item selecionado.
- **Exibir Refugo:** Define se mostrará ou não as informações referentes ao refugo.
