Relátorio de Recursividade de OPs
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório tem como objetivo indicar as Ordens de Produção que possuem recursividade, a tela pode ser acessada através do caminho abaixo:

**Menu principal -> Produção -> Relatórios de produção -> Relatório de recursividade de ordens de produção**

.. image:: /_static/BR\ One\ Produção/Relatorios/Relatório\ de\ recursividade\ de\ OPs/01.png
   :scale: 100%

No cabeçalho da tela ‘Relatório de recursividade de ordens de produção’, há três filtros que podem ser utilizados na busca das ordens de produção recursivas, são eles:

- **Data**
- **Nº do item**
- **Ordem de produção**

Cada um dos filtros citados acima possui algumas regras e especificidades, segue abaixo detalhes:

- **Data:**

É o único campo obrigatório da tela, caso não seja informado e o usuário tente realizar a procura das OPs recursivas, o add-on irá retornar a seguinte mensagem:

.. image:: /_static/BR\ One\ Produção/Relatorios/Relatório\ de\ recursividade\ de\ OPs/02.png
   :scale: 100%

**BR One :: O filtro de data é obrigatório.**

Neste filtro é permitido inserir um período de no máximo 31 dias,  caso usuário insira um período maior, o add-on irá retornar a seguinte mensagem:

.. image:: /_static/BR\ One\ Produção/Relatorios/Relatório\ de\ recursividade\ de\ OPs/03.png
   :scale: 100%
  
**BR One :: O filtro de data tem um período superior a 31 dias.**

**Exemplo:**

Este filtro permite selecionar um intervalo de itens, ao aplicar o filtro, serão retornados apenas as OPs recursivas que façam parte do intervalo selecionado.
