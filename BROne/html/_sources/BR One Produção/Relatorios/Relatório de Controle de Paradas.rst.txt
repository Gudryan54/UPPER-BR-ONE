Relatório de Controle de Paradas
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório retorna uma lista do consumo geral dos itens por Ordem de Produção, conforme horizonte estipulado nos filtros.

.. image:: /_static/BR\ One\ Produção/Relatorios/Controle\ de\ Paradas/13.png

- **Data Inicial:** Define a data inicial da análise.
- **Data Final:** Define a data final da análise.
- **Cód. Produto:** Define um código de produto para a análise.
- **Inserir OP:** Define uma seleção de OPs para análise.
- **Inserir Depósito:** Define um depósito para a análise.
