Centro de Trabalho
~~~~~~~~~~~~~~~~~~~~~~

Para acessar o cadastro de Centro de Trabalho é necessário ir  no menu: 

**Administração -> Definição -> Produção -> Centro de Trabalho**

.. image:: /_static/BR\ One\ Produção/Cadastros/Centro\ de\ Trabalho/image_2023_07_25T13_11_08_706Z.png
   :scale: 80%

A tela é aberta no modo de Procurar, para adicionar deve ativar o botão Adicionar ou Ctrl+A, a tela ficará no modo de adição:

.. image:: /_static/BR\ One\ Produção/Cadastros/Centro\ de\ Trabalho/Aspose.Words.f5a48aa2-e57e-48d8-a80a-020305dce230.003.png
   :scale: 80%

Para adição dos dados é necessário informar o codigo, descrição e calendário:

No campo o calendário serão listados todos calendários cadastrados.

.. image:: /_static/BR\ One\ Produção/Cadastros/Centro\ de\ Trabalho/Aspose.Words.f5a48aa2-e57e-48d8-a80a-020305dce230.004.png
   :scale: 80%

Após inserção de todas as informações clicar em "Adicionar".

.. image:: /_static/BR\ One\ Produção/Cadastros/Centro\ de\ Trabalho/Aspose.Words.f5a48aa2-e57e-48d8-a80a-020305dce230.005.png
   :scale: 80%

Esse cadastro deverá ser único por empresa, pode ser gerenciado por filial caso tenha recursos separados.

É apenas um cadastro gerencial, e utilizado para recuperção do custo hora no Simulador de custos, caso não tenha um custo hora no grupo de recurso.
