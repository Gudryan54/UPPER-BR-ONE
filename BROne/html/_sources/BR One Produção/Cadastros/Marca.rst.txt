Marca
~~~~~~~~~~

Para acessar o cadastro de Marca é necessário ir  no menu: 

**Administração -> Definição -> Produção -> Marca**

.. image:: /_static/BR\ One\ Produção/Cadastros/Marca/Aspose.Words.19e83c97-4418-48cf-8af8-8bd0f0862b09.003.png
   :scale: 80%
Na coluna "Code" Informe o código da marca.

.. image:: /_static/BR\ One\ Produção/Cadastros/Marca/Aspose.Words.19e83c97-4418-48cf-8af8-8bd0f0862b09.004.png
   :scale: 80%
E na coluna "Name" informe a descrição da marca.

.. image:: /_static/BR\ One\ Produção/Cadastros/Marca/Aspose.Words.19e83c97-4418-48cf-8af8-8bd0f0862b09.005.png
   :scale: 80%
Após informado os dados selecione "Atualizar". 

.. image:: /_static/BR\ One\ Produção/Cadastros/Marca/Aspose.Words.19e83c97-4418-48cf-8af8-8bd0f0862b09.006.png
   :scale: 80%
Esse cadastro deve ser realizado linha a linha e essas infomações serão listadas no cadastro de Recurso.
