Atualização de Custo/Hora
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Para acessar a tela de Atualização de Custo/Hora de componentes é necessário ir no menu:


**Produção -> Manutenção -> Atualização de Custo/Hora**

Nesta tela é possível atualizar o valor/hora de todos os grupos de recurso ou centro de trabalho.

.. image:: /_static/BR\ One\ Produção/Processo/Manutenção/Atualização\ de\ Custo\ Hora/Aspose.Words.1da940b5-cd50-479c-8156-bb9d2bb15047.001.png
   :scale: 80%


Os tipos de atualização que podem ser feitos são atualização de custo de Centro de Trabalho e de Grupo de Recursos.

.. image:: /_static/BR\ One\ Produção/Processo/Manutenção/Atualização\ de\ Custo\ Hora/Aspose.Words.1da940b5-cd50-479c-8156-bb9d2bb15047.002.jpeg
   :scale: 80%

De acordo com o tipo de atualização selecionado, serão mostrados todos os centros de trabalhos ou grupo de recursos cadastrados.


.. image:: /_static/BR\ One\ Produção/Processo/Manutenção/Atualização\ de\ Custo\ Hora/Aspose.Words.1da940b5-cd50-479c-8156-bb9d2bb15047.003.png
   :scale: 80%

.. image:: /_static/BR\ One\ Produção/Processo/Manutenção/Atualização\ de\ Custo\ Hora/Aspose.Words.1da940b5-cd50-479c-8156-bb9d2bb15047.004.png
   :scale: 80%


A atualização de valores pode ser feita de um por um ou informando uma porcentagem para atualizar todos de uma vez.

A porcentagem pode ser tanto positiva, para aumentar os valores, quanto negativa, para diminuí-los.


.. image:: /_static/BR\ One\ Produção/Processo/Manutenção/Atualização\ de\ Custo\ Hora/Aspose.Words.1da940b5-cd50-479c-8156-bb9d2bb15047.005.png
   :scale: 80%

.. image:: /_static/BR\ One\ Produção/Processo/Manutenção/Atualização\ de\ Custo\ Hora/Aspose.Words.1da940b5-cd50-479c-8156-bb9d2bb15047.006.png
   :scale: 80%


Há duas validações que impedem a atualização:

- Se o valor atual for diferente de 0 e o novo valor for 0 ou menor que 0
- Se o valor atual for 0 e o novo valor for menor que 0

Para as duas validações, a seguinte mensagem de erro será exibida:

.. image:: /_static/BR\ One\ Produção/Processo/Manutenção/Atualização\ de\ Custo\ Hora/Aspose.Words.1da940b5-cd50-479c-8156-bb9d2bb15047.007.png
   :scale: 80%

*Erro na linha x. O valor na coluna Novo valor deve ser maior que zero.*
