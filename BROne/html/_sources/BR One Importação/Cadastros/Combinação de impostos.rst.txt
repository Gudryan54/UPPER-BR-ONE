Combinação de impostos
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Para acessar a tela de **Combinação de impostos**, é necessatio ir ao menu:

**Admininstração -> Definição -> Finanças -> Imposto -> Configurar regra de imposto -> Combinação de impostos**

Deve ser marcado o flag “**Desp. adic. Importação”** para as combinações de impostos que forem participar do processo, para selecionar qual combinação será atribuída às colunas **Despesas adicionais código de imposto** na linha do item na NF de nacionalização (NF de entrada).
