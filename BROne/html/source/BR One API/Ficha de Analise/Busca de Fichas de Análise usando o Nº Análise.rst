Busca de Fichas de Análise usando o Nº Análise
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este endpoint realiza a busca das fichas de análise usando o campo '**numeroAnalise**', é possível filtrar as buscas usando os códigos de modelo de análise.
