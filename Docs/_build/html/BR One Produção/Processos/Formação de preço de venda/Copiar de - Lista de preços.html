Copiar de - Lista de preços

Ao selecionar para copiar de Lista de preços será aberta uma tela com filtros de item, Grupo de item, Código NCM, Grupo de materiais, Fabricante, Lista de preços e características.

É obrigatória a seleção de uma lista de preços. Caso não seja selecionada nenhuma lista de preços e seja clicado em OK, a seguinte mensagem será exibida:

BR One :: Campo “Lista de preços” é de preenchimento obrigatório.

Após selecionar uma lista de preços e os filtros desejados e clicar em OK, será exibida a tela com o resultado dos filtros. Basta selecionar a linha que deseja copiar para a formação e clicar em OK.

A tela Aguarde aparecerá e permanecerá até que todos os itens sejam carregados na tela.
