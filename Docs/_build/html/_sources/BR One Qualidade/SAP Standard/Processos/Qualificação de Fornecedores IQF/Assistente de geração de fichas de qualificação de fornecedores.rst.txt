Assistente de geração de fichas de qualificação de fornecedores
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Para acessar a tela de **Assistente de geração de fichas de qualificação de fornecedores**, é necessatio ir ao menu:

**Qualidade -> Assistente de Geração de Fichas de Qualificação de Fornecedores**

Nessa tela, é possível criar fichas de qualificação em lote para os fornecedores.

.. image:: /_static/BR\ One\ Qualidade/Processo/Qualificação\ de\ fornecedores/Aspose.Words.67881489-10be-41d8-9938-4848d61ce893.038.png
   :scale: 100%

Assim como na tela de ficha de qualificação, o campo **Fornecedor** lista apenas os fornecedores que possuem modelo padrão, que estão ativos e não tem nenhuma ficha em aberto com status planejado ou liberado.

O campo **Modelo** lista os modelos de qualificação cadastrados como padrão no cadastro do fornecedor. O range escolhido no campo Modelo de Qualificação impactar na exibição dos resultados. É possível notar abaixo que no range do fornecedor está selecionado o fornecedor F0001, mas no campo abaixo o modelo padrão deste fornecedor não foi selecionado, desta forma ele não aparecerá na lista para a geração da ficha. 

.. image:: /_static/BR\ One\ Qualidade/Processo/Qualificação\ de\ fornecedores/Aspose.Words.67881489-10be-41d8-9938-4848d61ce893.039.png
   :scale: 100%

Para aparecer na lista é preciso que tanto o **fornecedor** esteja dentro do range quanto o **modelo de qualificação** padrão do fornecedor também esteja no range selecionado do campo Modelo de qualificação.

Para gerar as fichas, devem ser selecionadas todas as linhas e clicar no botão **Gerar**. 
Ou caso queira selecionar apenas para alguns fornecedores selecionar apenas os fornecedores desejados e clicar em **Gerar**.
