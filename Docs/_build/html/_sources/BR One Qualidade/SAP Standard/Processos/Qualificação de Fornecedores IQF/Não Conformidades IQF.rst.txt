Não conformidades – IQF
~~~~~~~~~~~~~~~~~~~~~~~~~~

Para acessar a tela de **Não conformidades – IQF**, é necessatio ir ao menu:

**Administração -> Definição -> Qualidade > Não conformidades – IQF**

Nesta tela, devem ser cadastradas todas as não conformidades existentes. 

.. image:: /_static/BR\ One\ Qualidade/Processo/Qualificação\ de\ fornecedores/Aspose.Words.67881489-10be-41d8-9938-4848d61ce893.003.png
   :scale: 100%

Não é possível inserir várias linhas ao mesmo tempo, pois para inserir uma nova linha a tela deve estar no modo Ok, como mostrado abaixo. 
É possível também eliminar linhas desde que a mesma não esteja vinculada a nenhuma especificação.

.. image:: /_static/BR\ One\ Qualidade/Processo/Qualificação\ de\ fornecedores/Aspose.Words.67881489-10be-41d8-9938-4848d61ce893.004.png
