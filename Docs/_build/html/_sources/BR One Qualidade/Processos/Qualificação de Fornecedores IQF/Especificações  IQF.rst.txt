Especificações – IQF
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Para acessar a tela de **Especificações - IQF** , é necessatio ir ao menu:

**Administração -> Definição -> Qualidade -> Especificações – IQF**

Nessa tela, é possível cadastrar/alterar as especificações e realizar o vínculo com uma ou mais não conformidades, lembrando que caso a especificação estiver sendo utilizada em algum modelo ela não poderá ser alterada.

.. image:: /_static/BR\ One\ Qualidade/Processo/Qualificação\ de\ fornecedores/Aspose.Words.67881489-10be-41d8-9938-4848d61ce893.005.png
   :scale: 100%

Os tipos de dados e tipos de resultado só poderão ser combinados da seguinte maneira:

- **Numérico:** Intervalo e Fixo
- **Data:** Intervalo e Fixo
- **Sim/Não:** Fixo

Existem algumas combinações que não são possíveis de cadastrar como por exemplo se o usuário tentar cadastrar um tipo de dado como **Sim/Não** e no campo **Tipo de resultado** tentar selecionar intervalo. 
Caso o usuário tente essas combinações não possíveis o sistema não deixará adicionar a especificação exibindo o motivo, como na mensagem abaixo:

.. image:: /_static/BR\ One\ Qualidade/Processo/Qualificação\ de\ fornecedores/Aspose.Words.67881489-10be-41d8-9938-4848d61ce893.006.png
   :scale: 100%

*BR One :: Só é possível utilizar o tipo de dado "Sim/Não" com o tipo de resultado "Fixo".*

Também é possível remover as especificações que não estejam vinculadas a nenhum modelo de qualificação conforme a simulação abaixo.
