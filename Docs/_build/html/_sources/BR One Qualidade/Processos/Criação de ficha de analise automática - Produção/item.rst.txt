Vínculo de Modelo de Análise no Item
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

No cadastro do item é possível configurar um ou mais modelos de análise, sua configuração deve ser feita através da aba **"Qualidade"** onde selecionarmos o modelo desejado no campo **"Modelo de análise"**.

Para inserir apenas um modelo de análise, basta informar o modelo desejado no campo **"Modelo de análise"**, conforme print abaixo:

.. image :: /_static/BR\ One\ Qualidade/Processo/Automatico\ Prod/Vínculo\ de\ Modelo\ de\ Análise\ no\ Item/Qualidade1.PNG


Caso deseje informar mais de um modelo de análise para um item, deve-se clicar no botão **"..."** e informar os modelos desejados:

.. image :: /_static/BR\ One\ Qualidade/Processo/Automatico\ Prod/Vínculo\ de\ Modelo\ de\ Análise\ no\ Item/Item132.PNG
