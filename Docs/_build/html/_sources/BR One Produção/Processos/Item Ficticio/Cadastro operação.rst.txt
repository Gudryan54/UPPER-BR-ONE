Cadastro da Operação fictício
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Para operações que serão utilizadas dentro de um Produto que é item fictício, apenas as operações que estiverem com o flag **Operação Fictícia** marcada, serão exibidas para seleção na criação dos roteiros.

.. image:: /_static/BR\ One\ Produção/Processo/ItemFicticio/img4.JPG
   :scale: 80%

Para itens que não são fictícios essas operações não serão exibidas, e dentro da Ordem de produção elas também não serão exibidas.
