Monitor de recursos
~~~~~~~~~~~~~~~~~~~~~~~~~~

Nesta tela é possivel monitorar os recursos e visualizar seus status em tempo real.

Para acessar a tela de Monitor de recursos é necessário ir  no menu:

**Produção -> Apontamentos -> Monitor de recursos**

.. image :: /_static/BR\ One\ Produção/Processo/Apontamento/Monitor\ de\ recursos/Aspose.Words.c922076d-37b1-4dfd-a582-1f924e309120.001.png
   :scale: 80%

Para filtrar os recursos exibidos, você tem as seguintes opções de filtro:

.. image :: /_static/BR\ One\ Produção/Processo/Apontamento/Monitor\ de\ recursos/Aspose.Words.c922076d-37b1-4dfd-a582-1f924e309120.002.png
   :scale: 80%

Exibir **todos**: Esta opção mostrará todos os recursos disponíveis, independentemente de estarem alocados em uma ordem de produção (OP) ou não.

Exibir apenas os recursos **alocados em OP**: Ao selecionar esta opção, somente os recursos que estão atualmente alocados em uma ordem de produção serão exibidos. 

Exibir apenas os recursos **não alocados**: Escolhendo essa opção, apenas os recursos que não estão atualmente alocados em nenhuma ordem de produção serão exibidos. 

Também foram inclusos os filtros **Nº OP**, **Nº Item**, **Grupo de recurso** e **Recurso**.

.. image :: /_static/BR\ One\ Produção/Processo/Apontamento/Monitor\ de\ recursos/Aspose.Words.c922076d-37b1-4dfd-a582-1f924e309120.003.png
   :scale: 80%

Quando usuário selecionar a opção **"Todos"** ou **"Alocados em OP"**, todos esses filtros ficarão habilitados.

.. image :: /_static/BR\ One\ Produção/Processo/Apontamento/Monitor\ de\ recursos/Aspose.Words.c922076d-37b1-4dfd-a582-1f924e309120.004.png
   :scale: 80%

Quando usuário selecionar a opção **“Não alocados”**, apenas os filtros **Grupo de recurso** e **Recurso** ficarão habilitados.

.. image :: /_static/BR\ One\ Produção/Processo/Apontamento/Monitor\ de\ recursos/Aspose.Words.c922076d-37b1-4dfd-a582-1f924e309120.005.png
   :scale: 80%

No filtro **Nº OP**, serão carregadas apenas as Ordens de produção (OPs) que estejam com status liberadas.

No filtro **Nº Item**, serão carregados apenas os itens que são de produção e estejam ativos.

No filtro **Recurso**, serão carregados apenas os recursos ativos.

Além disso, é possível ajustar a taxa de atualização da tela. Para fazer isso, é necessário interromper o monitoramento atual, escolher a taxa de atualização desejada e, em seguida, reiniciar o monitoramento novamente.
