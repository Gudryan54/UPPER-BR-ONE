
Definições de Unidades de Medidas
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Na tela de **Definições de Unidades de Medidas** será configurado se o retorno das quantidades calculadas serão fracionados ou não.

Podemos ter unidades de medida que não possam ser fracionadas, como é o caso de **peça**, porém possuímos casos em que as unidades de medida possam ser fracionadas como é o caso de **quilos**. 

**Administração -> Definição -> Estoque -> Unidades de medida.**
