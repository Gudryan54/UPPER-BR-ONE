Relatório Parada de Recursos
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório retorna uma lista com as paradas de recursos, o retorno é realizado de acordo com os filtros selecionados.

.. image:: /_static/BR\ One\ Produção/Relatorios/Parada\ de\ Recursos/21.png

- **Recurso:** Define o tipo de recurso.
- **Status:** Define  motivo da parada para a analise
- **Data Inicial:** Define qual o início da busca para analise
- **Data Final:** Define qual o fim da busca para análise.
