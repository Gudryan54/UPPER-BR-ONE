Relatório de Recomendações do MRP
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório retorna uma lista das recomendações dos cenários MRP, o retorno é realizado de acordo com os filtros selecionados.

.. image:: /_static/BR\ One\ Produção/Relatorios/Recomendações\ do\ MRP/25.png

- **Cenário MRP:** Define o cenário MRP que será retornado.
- **Código do Item:** Define um item que será retornado.
- **Tipo de pedido:** Define o tipo de pedido ‘Ordem de produção’ ou ‘Solicitação de compra’.
- **Fornecedor Preferencial:** Define o fornecedor preferencial.
