Relatório de Tempo Previsto x Realizado
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório retorna uma lista das OPS comparando o tempo previsto com o realizado, o retorno é feito de acordo com os filtros selecionados.

.. image:: /_static/BR\ One\ Produção/Relatorios/Tempo\ Previsto\ x\ Realizado/35.png

- **Inserir Recurso:** Define os Recursos que serão retornados.
- **Inserir o tipo de data:** Define se o tipo de data analisado será data de lançamento, liberação, vencimento ou fechamento.
- **Data Inicial e Data Final:** Define o intervalo de datas que será analisado.
- **Inserir Item:** Define os itens que serão retornados.
- **Inserir Grupo de recurso:** Define grupos de recurso que serão retornados.
- **Selecione a forma de Agrupamento:** Define o tipo de agrupamento utilizado no retorno.
- **Mostrar Detalhe?:** Define se o retorno mostrará os detalhes das OPs.
