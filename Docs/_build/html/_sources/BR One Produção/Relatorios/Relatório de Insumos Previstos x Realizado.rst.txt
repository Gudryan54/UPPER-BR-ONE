Relatório de Insumos Previstos x Realizado
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório retorna uma lista comparando o Previsto x Realizado dos insumos da OP, o retorno é realizado de acordo com os filtros selecionados.

.. image:: /_static/BR\ One\ Produção/Relatorios/Insumos\ Previstos\ x\ Realizado/19.png

- **Tipo de Data:** Define se o tipo de data analisado será data de lançamento, liberação, vencimento ou fechamento.
- **Data Inicial e Data Final:** Define o intervalo de datas que será analisado.
- **Mostrar Detalhe?:** Define se o relatório irá retornar os componentes.
- **Inserir Item PA:** Define se será retornado os PAs selecionados.
- **Inserir Componente:** Define se será retornado os componentes selecionados.
- **Selecione a Forma de Agrupamento:** Pode ‘Agrupar por Ordem de Produção’ ou ‘Agrupar por Itens’.
- **OP Inicial:** Define o início que será levado ao relatório.
- **OP Final:** Define a lista de OPs que serão analisadas.
