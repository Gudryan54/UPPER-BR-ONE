Relatório de Status da Ordem de Produção
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este relatório retorna uma lista das OPS existentes, o retorno é realizado de acordo com os filtros selecionados.

.. image:: /_static/BR\ One\ Produção/Relatorios/Status\ da\ Ordem\ de\ Produção/33.png

- **Inserir o tipo de data:** Define se o tipo de data analisado será data de lançamento, liberação, vencimento ou fechamento.
- **Data Inicial e Data Final:** Define o intervalo de datas que será analisado.
- **Inserir Item PA: Define se será retornado os PAs selecionados.
- **Inserir depósito: Define depósito que será retornado.
- **Inserir status da OP:** Defini o status desejado das OPs retornadas.
