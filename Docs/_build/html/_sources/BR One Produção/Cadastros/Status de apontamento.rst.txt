Status de apontamento
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Para acessar o Cadastro de Status do apontamento é necessário ir  no menu:

**Administração -> Definição -> Serviço -> Status do apontamento**

.. image:: /_static/BR\ One\ Produção/Processo/Ordem\ de\ Manutenção/Status\ de\ apontamento/Aspose.Words.27c7ada7-0c00-4530-b552-61ace4bcdfce.001.png
   :scale: 80%

Na coluna "Code" Informe o código do status de apontamento e na coluna "Name" informe a descrição do status de apontamento.

Esse cadastro deve ser realizado linha a linha e os cadastros realizados nesta tela serão utilizados ao realizar apontamento de ordem de manutenção, é possível preencher o status para cada linha de apontamento, lembrando que o campo não é de preenchimento obrigatório.

   :scale: 80%
