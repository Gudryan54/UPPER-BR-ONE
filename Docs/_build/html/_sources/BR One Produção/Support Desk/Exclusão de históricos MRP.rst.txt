Exclusão de histórico MRP
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Para acessar a funcionalidade **‘BR One Produção: Exclusão de histórico MRP’** é necessário ir  no menu:

**Ajuda -> Support Desk -> Restaurar -> BR One -> BR One Produção: Exclusão de histórico MRP**

.. image:: /_static/BR\ One\ Produção/Suporte\ Desk/Exclusão\ de\ históricos\ MRP/Aspose.Words.147ff8a6-6bc7-42cb-b98a-0d19cdb39fbb.002.png
   :scale: 80%

Essa tela tem como objetivo realizar a exclusão dos históricos do MRP. Ao clicar na funcionalidade, será aberta a tela abaixo: 

.. image:: /_static/BR\ One\ Produção/Suporte\ Desk/Exclusão\ de\ históricos\ MRP/Aspose.Words.147ff8a6-6bc7-42cb-b98a-0d19cdb39fbb.003.png
   :scale: 80%

No formulário será possível informar o cenário desejado e até qual data de execução os históricos deverão ser excluídos.

Lembrando que se excluídos, os históricos não poderão mais ser recuperados.

Portanto, deve-se estar certo antes de utilizar esta funcionalidade.

Ao clicar em **"Remover"** será exibido a mensagem de confirmação abaixo:

.. image:: /_static/BR\ One\ Produção/Suporte\ Desk/Exclusão\ de\ históricos\ MRP/Aspose.Words.147ff8a6-6bc7-42cb-b98a-0d19cdb39fbb.004.png
   :scale: 80%

*BR One :: Será(ão) excluído(s) X histórico(s). O Processo é irreversível. Deseja continuar?*

Caso usuário selecione **‘Sim’** o histórico será excluído, caso usuário clique em **"Não"**, o processo será cancelado. Se não houver históricos até a data de execução informada, ao clicar em ‘Remover’ exibirá a mensagem abaixo:

.. image:: /_static/BR\ One\ Produção/Suporte\ Desk/Exclusão\ de\ históricos\ MRP/Aspose.Words.147ff8a6-6bc7-42cb-b98a-0d19cdb39fbb.005.png
   :scale: 80%

*BR One :: Não existem históricos até a data escolhida ou os mesmos já foram excluídos.*
