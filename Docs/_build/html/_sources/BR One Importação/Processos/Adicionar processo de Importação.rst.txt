Adicionar Processo de Importação
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Para começar o processo de importação, devemos adicionar um novo documento. Para isto, preencha os dados obrigatórios do cabeçalho como Fornecedor, Despachante, Moeda e, no caso de Multi-filiais, indicar qual será a filial.  Após preenchimento correto das informações, clique no botão **adicionar** no rodapé para prosseguir com o processo.

.. image:: /_static/BR\ One\ Importação/Processo/Adicionar\ processo\ de\ Importação/1.PNG
   :scale: 80%

- **Possíveis ocorrências durante o processo**

Não será possível adicionar nenhum Processo de importação sem dados. Ao clicar em *Adicionar* nesse cenário, a seguinte mensagem será exibida:

.. image:: /_static/BR\ One\ Importação/Processo/Adicionar\ processo\ de\ Importação/Aspose.Words.c6eed0be-a92d-4791-b70e-b4d6d464cf4d.002.jpeg
   :scale: 80%

*BR One :: O fornecedor não foi informado.*

Nenhuma operação estará disponível sem a seleção de um fornecedor e despachante. Se o usuário tentar realizar algum processo sem selecionar um fornecedor e despachante, a seguinte mensagem será exibida:

.. image:: /_static/BR\ One\ Importação/Processo/Adicionar\ processo\ de\ Importação/Aspose.Words.c6eed0be-a92d-4791-b70e-b4d6d464cf4d.003.jpeg
   :scale: 80%

*BR One :: O fornecedor e o despachante devem ser informados.*

Se a moeda não for informada, ao tentar realizar algum processo de importação, a seguinte mensagem será exibida:

.. image:: /_static/BR\ One\ Importação/Processo/Adicionar\ processo\ de\ Importação/Aspose.Words.c6eed0be-a92d-4791-b70e-b4d6d464cf4d.004.png
   :scale: 80%

*BR One :: A moeda não foi informada.*

Também não será possível realizar nenhuma ação na tela sem que o formulário esteja adicionado/atualizado. Caso o usuário tente iniciar o processo com o formulário em modo Adicionar/Atualizar, a seguinte mensagem será exibida:

.. image:: /_static/BR\ One\ Importação/Processo/Adicionar\ processo\ de\ Importação/Aspose.Words.c6eed0be-a92d-4791-b70e-b4d6d464cf4d.005.jpeg
   :scale: 80%

*BR One :: Insira/atualize o documento antes de continuar.*

Caso a filial não seja informada, ao tentar realizar algum processo de importação, a seguinte mensagem será exibida:

.. image:: /_static/BR\ One\ Importação/Processo/Adicionar\ processo\ de\ Importação/Aspose.Words.c6eed0be-a92d-4791-b70e-b4d6d464cf4d.006.jpeg
   :scale: 80%

*BR One :: A Filial não foi informada.*

Caso o usuário tente iniciar o processo de importação sem as configurações de importação terem sido realizadas, a seguinte mensagem será exibida:

.. image:: /_static/BR\ One\ Importação/Processo/Adicionar\ processo\ de\ Importação/Aspose.Words.c6eed0be-a92d-4791-b70e-b4d6d464cf4d.007.jpeg
   :scale: 80%

*BR One :: Para realizar o processo, é necessário antes realizar todas as configurações de importação.*


- **Detalhes de configurações necessárias**

O preenchimento do campo **Fornecedor** será realizado apenas com parceiros de negócios do tipo **Fornecedor** e que possuam em seu cadastro o campo **Moeda** igual à **Todas as moedas** e cujo país definido seja diferente de Brasil (será considerado o país do endereço padrão).


.. image:: /_static/BR\ One\ Importação/Processo/Adicionar\ processo\ de\ Importação/3.png


Para bases Multi-filiais, no campo *Filial* deve ser selecionada a filial ativa no momento.  Apenas aparecerão as filiais que estão atribuídas ao usuário logado no momento. Se na atribuição de filial do usuário alguma filial não esteja com o flag **Atribuído** marcado, ela não aparecerá para seleção.
