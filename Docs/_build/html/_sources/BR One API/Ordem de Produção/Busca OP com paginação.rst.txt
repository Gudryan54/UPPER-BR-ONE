Busca Ordens de Produção
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Este endpoint realiza a busca a todas ordens de produção existentes na base. 

Ela pode ser realizada utilizando o campo '**tipo**', '**status**' e '**numero**', já o campo '**pagina**' define quantas páginas serão retornadas na API após a busca, e o campo '**quantidade**' indica quantas OPs serão exibidas por página.
