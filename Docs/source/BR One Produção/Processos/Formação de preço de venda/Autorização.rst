Autorização
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Para o usuário ter acesso à tela "Formação de preços de vendas", é necessário que ele tenha autorização total para a autorização "Formação de preços de vendas". A tela "Autorizações" pode ser acessada pelo caminho *Administração > Inicialização do sistema > Autorizações > Autorizações gerais.*
